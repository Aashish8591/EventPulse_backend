# 🚀 EventPulse Backend

Spring Boot REST API for Event Feedback Management System.

## ✨ Features

- Event CRUD APIs
- Feedback CRUD APIs
- Search Feedback
- MySQL Integration
- REST APIs

## 🛠 Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL

## ▶️ Run

```bash
mvn spring-boot:run
```

or run **EventPulseApplication.java**
